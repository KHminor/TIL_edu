# 1. 데드락이란? 

# 2. 데드락의 4가지 조건, 4가지 이상의 해결 방법을 이야기 하시오.





















































답 

- 데드락이란?

  두개 이상의 스레드가 하나의 공유된 자원을 놓고 서로 경쟁하는 상황.





- 데드락의 4가지 조건

1. mutual exclution (상호 배제)
2. no preeption (비선점)
3. hold and wait (점유와 대기)
4. circular wait (환형 대기)



- 데드락의 해결 방법

1. 예방
2. 회피
3. 탐지
4. 회복
5. 무시
